# MkDocs 资源导航项目维护说明

## 快速使用
1. 克隆项目：`git clone https://github.com/chiangprc/mkdocs-site.git`
2. 安装依赖：`pip install -r requirements.txt`
3. 本地运行：`mkdocs serve`
4. 发布更新：`mkdocs gh-deploy --force`

## 文件结构
- docs/         # 所有内容文件
- mkdocs.yml    # 配置文件
- .nojekyll     # 禁用Jekyll
- requirements.txt  # 依赖清单
